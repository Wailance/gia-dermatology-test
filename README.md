# Тест по дерматовенерологии (ГИА)

Интерактивный тренажёр с 100 вопросами для подготовки к государственной итоговой аттестации по дерматовенерологии.

## Возможности

- Прохождение теста в исходном или случайном порядке вопросов
- Навигация по вопросам
- Подсчёт результата и разбор ошибок

## Локальный запуск

```bash
npx serve .
```

Откройте http://localhost:3000

## Обновление вопросов из PDF

```bash
npm run build:data
```
